Privacy Policy — English

Who we are

Cristian Taraborrelli Studio SRL — Italian VAT 17597231004 — operates cristiantaraborrellistudio.net and associated applications (including Pinterest, Instagram, LinkedIn integrations used to publish the artist's own portfolio).

What we collect

  • Website visitors. No third-party tracking cookies. GitHub Pages hosting logs standard server data (anonymised IP, user agent, requested page) for technical security and diagnostics only.
  • Pinterest / social applications. Internal apps (e.g. Cristian Taraborrelli Studio registered on developers.pinterest.com) access only the owner's Pinterest account via OAuth and official APIs, for automated publishing of the owner's Pins (titles, descriptions, links back to this website). They do not collect data on other users, do not read followers, and do not scrape.

How we use data

  • Project metadata (titles, descriptions, photo credits) is published publicly as part of the artist's portfolio.
  • Direct email contacts are used only to reply to received requests; never added to marketing lists.
  • We do not share data with third parties for advertising.

Storage

Social API access tokens are stored locally (in a .env file on the owner's computer) and never transmitted to third-party servers. Server logs (GitHub Pages) follow GitHub's own policy.
